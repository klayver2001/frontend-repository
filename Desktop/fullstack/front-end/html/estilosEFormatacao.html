Estilos com HTML

O atributo HTML <style> usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

Tem a seguinte sintaxe : 

                    
                        < style="property:value;">

Cor de fundo

A background-colorpropriedade CSS define a cor de fundo de um elemento HTML. 


                style=" background-color:powderblue; "
                style=" background-color:tomato; "

Cor do texto

A color propriedade CSS define a cor do texto para um elemento HTML: 


                style=" color:blue; "
                style=" color:red; "

Fontes

A propriedade CSS font-family define a fonte a ser usada para um elemento HTML: 


                style=" font-family:verdana; "
                style=" font-family:courier; "

Tamanho do texto

A font-size propriedade CSS define o tamanho do texto para um elemento HTML: 


                style="font-size:300%;"
                style="font-size:160%;"

Alinhamento de texto

A text-alignpropriedade CSS define o alinhamento horizontal do texto para um elemento HTML:


                style="text-align:center;"

Resumo do Capítulo

  • Use o style atributo para estilizar elementos HTML
  • Usar background-color para cor de fundo
  • Usar color para cores de texto
  • Usar font-family para fontes de texto
  • Usar font-size para tamanhos de texto
  • Usar text-align para alinhamento de texto

Formatação de texto HTML

HTML contém vários elementos para definir texto com um significado especial.

Exemplo: 

            This text is bold

            This text is italic

            This is subscript and superscript
