Os atributos HTML fornecem informações adicionais sobre elementos HTML. Atributos HTML Todos os elementos HTML podem ter atributos Os atributos fornecem informações adicionais sobre os elementos Os atributos são sempre especificados na tag inicial Os atributos geralmente vêm em pares nome/valor como: nome="valor"

Ex: A tag "a" define um hyperlink.
O "href"atributo especifica a URL da página para a qual o link vai:

Outros atributos :
  • O atributo src : Especifica o caminho para a imagem a ser exibida

  • Os atributos de largura e altura:
    A tag "img" também deve conter os atributos widthe height, que especificam a largura e a altura da imagem (em pixels):

  • O atributo alt : Especifica um texto alternativo para uma imagem, se a imagem por algum motivo não puder ser exibida.

  • O atributo de estilo : "style" atributo é usado para adicionar estilos a um elemento, como cor, fonte, tamanho e muito mais.

  • O atributo "lang" : Serve para declarar o idioma da página da Web. Isso é para auxiliar mecanismos de busca e navegadores.

  • O atributo do título : "title" define algumas informações extras sobre um elemento.
    O valor do atributo title será exibido como uma dica de ferramenta quando você passar o mouse sobre o elemento:
